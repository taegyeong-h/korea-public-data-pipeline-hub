# 클라이언트 사이드 방식 (DBeaver의 '데이터 가져오기' / 파이썬 / psql \copy)

지금 화면에 엑셀(CSV)을 띄워놓으신 PC가 호스트 OS(내 컴퓨터)이고, DBeaver도 거기에 설치되어 있다면 게스트 OS에 파일을 옮길 필요가 전혀 없습니다.

DBeaver에서 192.168.56.10으로 접속합니다.

앞서 만든 ods.tm_subway_station_master 테이블을 마우스 우클릭합니다.


엑셀파일 CSV UTF-8(쉼표로 분리)(*.csv) 로 변환

[데이터 가져오기 (Import Data)]를 누릅니다.

이때 탐색기 창이 뜨면, 호스트 OS(내 컴퓨터)에 있는 CSV 파일을 선택해 줍니다.

DBeaver가 알아서 내 컴퓨터의 CSV를 읽어 네트워크를 통해 VirtualBox 안의 Postgres로 초고속 주입을 해줍니다.
